# 📖 Overview

Social Network API is a scalable backend system inspired by modern social media platforms.

The project is designed using Django and Django REST Framework with a focus on clean architecture, maintainability, scalability, security, and production-ready development practices.

It provides a complete backend infrastructure for handling users, social interactions, content creation, engagement systems, messaging, notifications, media management, and real-time communication.

# 🚀 Getting Started

```bash
git clone https://github.com/a3ahmadi/social_network.git

cd social_network

python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
